## 📖 Overview

**FragShare** is an advanced Real World Asset (RWA) protocol that tokenizes premium Indian real estate (Commercial, Residential, Office) and integrates it directly into a fully composable DeFi ecosystem on the Solana blockchain.

Unlike traditional platforms that merely issue SPL tokens representing fractions of a property, FragShare builds a complete, highly-liquid financial stack around these assets, unlocking unprecedented capital efficiency for investors.

## 🚀 Key Protocol Features

### 1. 🪙 Yield-Adjusted Automated Market Maker (AMM)
Real estate tokens suffer from low liquidity. FragShare solves this via a custom Constant Product AMM (`programs/fragshare_amm`) that programmatically adjusts the pricing curve based on the continuous rental yield generated by the underlying physical asset.
- **Instant Liquidity**: Swap USDC for property fractions instantly.
- **Fair Value Pricing**: The AMM utilizes Switchboard Oracles to track RERA status and NAV updates.

### 2. 🏦 Collateralized Debt Positions (CDPs)
Investors shouldn't have to sell their appreciating real estate to access liquidity. The FragShare Lending Protocol (`programs/fragshare_lending`) allows users to lock their fractional ownership into smart contract Vaults to mint **FragRupee (FRUP)**, an over-collateralized stablecoin pegged to the INR.
- **Max LTV**: 60% with an 80% Liquidation Threshold.
- **Stability Fee**: 5% APR algorithmically accrued.
- **Liquidations**: Dutch-auction liquidation engine executed fully on-chain.

### 3. 📦 ZK State Compression for Ownership
To reduce the exorbitant rent costs of managing 100,000+ individual property fraction accounts, FragShare utilizes **Concurrent Merkle Trees**.
- Ownership states are hashed into a single 32-byte root hash on-chain.
- Investors submit Zero-Knowledge proofs generated via an off-chain RPC indexer to execute swaps or claim yield.
- **Cost Reduction**: Reduces state rent by 99.99%.

### 4. 📈 Diversified Index Funds
A separate primitive that aggregates underlying property fractions into sector-specific indices (e.g., *Mumbai Commercial Real Estate Index*, *Bengaluru Tech Hub Index*), allowing retail users one-click exposure to diversified real-world yield.

The repository is built using a production-grade **Anchor Workspace** for the on-chain programs and a highly optimized **React (Vite)** dashboard for the client client interface.

```text
fragshare-monorepo/
│
├── programs/                      # Solana Smart Contracts (Rust)
│   ├── fragshare_amm/             # Yield-Adjusted AMM Program
│   └── fragshare_lending/         # CDP & Stablecoin Minting Program

├── src/                           # Client Application (React/Vite)

│   ├── index.css                  # Custom RWA UI Design System
│   └── ...

```

## 💻 Running the Project Locally

### 1. Launch the Frontend Application
The protocol's dynamic dashboard is fully functional and simulates the complex state management of the blockchain logic in real-time.

```bash
# Install dependencies
npm install

# Start the Vite development server
npm run dev
```
Navigate to `http://localhost:5173` to explore the AMM, mint FRUP debt, and claim simulated yield.

### 2. Build the Anchor Smart Contracts
To compile the `fragshare_amm` and `fragshare_lending` programs, ensure you have the Solana Toolchain and Anchor (`avm`) installed. (For Windows users, we recommend using WSL 2).

```bash
# Verify Anchor is installed
anchor --version

# Build the Rust smart contracts
anchor build
```

## 🔐 Security & Oracles
FragShare relies on distributed oracle networks (e.g., Switchboard) to fetch off-chain property valuations and legal statuses. The Lending program enforces strict arithmetic safety utilizing `checked_add`, `checked_mul`, and `checked_div` to prevent arithmetic overflow vulnerabilities.
